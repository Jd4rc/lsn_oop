# Изучение ООП на Python

Репозиторий с учебными задачами и упражнениями для изучения объектно-ориентированного программирования (ООП) на Python.

Цель репозитория — освоить основные принципы ООП на практике, решать тренировочные задачи и развивать навыки написания чистого, понятного и поддерживаемого кода.

## Изучаемые темы

В процессе работы над проектом рассматриваются:

- классы и объекты;
- инкапсуляция;
- наследование;
- полиморфизм;
- композиция и агрегация;
- абстракция;
- свойства (`@property`);
- магические методы;
- итераторы;
- тестирование;
- статическая типизация.

## Реализованный функционал

### Product

- создание объектов товара;
- валидация цены;
- подтверждение снижения цены пользователем;
- объединение одинаковых товаров через метод `new_product`;
- получение строкового представления объекта;
- вычисление общей стоимости товара (`цена × количество`);
- поддержка магических методов:
  - `__str__`;
  - `__repr__`;
  - `__add__`.

### Category

- создание категорий товаров;
- подсчет общего количества категорий;
- подсчет общего количества товаров;
- добавление товаров в категорию;
- получение общего количества единиц товара через свойство `total_quantity`;
- поддержка итерации по товарам категории;
- поддержка магических методов:
  - `__len__`;
  - `__str__`.

### CategoryIterator

Реализован отдельный класс-итератор для последовательного перебора товаров категории с использованием протокола итерации Python.

## Установка

Клонируйте репозиторий:

```bash
git clone <https://github.com/Jd4rc/lsn_oop.git>
cd lsn-oop
```

Установите зависимости:

```bash
poetry install --with develop
```

Активируйте виртуальное окружение:

```bash
poetry shell
```

## Тестирование

Запуск всех тестов:

```bash
poetry run pytest
```

Получение отчета о покрытии:

```bash
poetry run pytest --cov=src --cov-report=term-missing --cov-report=html
```

После выполнения команды HTML-отчет будет доступен по пути:

```text
htmlcov/index.html
```
